# Gym Typescript

Gym Typescript is a landing page built using Vite and TypeScript, designed to provide an engaging and responsive user experience for fitness enthusiasts. This template provides a minimal setup to get React working in Vite with Hot Module Replacement (HMR) and ESLint integration.

## Features

- **Fast Development**: Utilizes Vite for a lightning-fast development environment.
- **Type Safety**: Written in TypeScript, ensuring type safety and enhancing code quality.
- **Responsive Design**: The landing page is fully responsive, providing an optimal viewing experience across various devices.
- **Custom Media Query Hook**: Implements a `useMediaQuery` hook to detect media query matches and update the component state dynamically based on screen size.
- **Scroll Handling**: Detects the scroll position to update the navbar state and highlight the current section.
- **Smooth Scrolling**: Implemented smooth scrolling for seamless navigation within the page using `react-anchor-link-smooth-scroll`.
- **Action Buttons**: Provides reusable `ActionButton` components for easy navigation to different sections of the page, enhancing user experience and interaction.
- **Animations**: Incorporates animations and transitions using `framer-motion` for a more dynamic user experience.
- **Form Handling**: Utilizes `react-hook-form` for efficient form handling and validation.
- **Section Components**: Includes various sections like Home, Benefits, Our Classes, Contact Us, and Footer, providing a comprehensive overview of the services offered.

## Tech Stack

- **Vite**: A build tool that provides a fast development environment and optimized production builds.
- **TypeScript**: A typed superset of JavaScript that improves developer experience and code quality.
- **React**: A JavaScript library for building user interfaces.
- **Tailwind CSS**: A utility-first CSS framework for building custom designs without leaving your HTML.
